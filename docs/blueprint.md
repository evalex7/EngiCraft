# **App Name**: RevitCraft Navigator

## Core Features:

- Hotkeys Guide: Відображає довідник гарячих клавіш Revit з можливістю пошуку для покращення швидкості робочого процесу.
- Workflow Tips: Надає покрокові інструкції та найкращі практики для типових задач у Revit.
- Personalized Recommendations Tool: Модуль на основі штучного інтелекту, який вивчає взаємодію користувача з додатком та пропонує функції і навчальні матеріали для покращення навичок роботи з Revit, враховуючи індивідуальний прогрес навчання та бажаний стиль роботи.
- YouTube Integration: Вбудовує відповідні відео-уроки з YouTube в додаток.
- Timestamped Video Playback: Дозволяє користувачам починати відтворення відео з YouTube з конкретних часових міток, щоб зосередитися на певних техніках. Приймає текстові запити користувача для визначення, з якого моменту потрібно запустити відео.
- Custom Content Addition: Дозволяє користувачам додавати власні нотатки, поради та посилання для подальшого використання.

## Style Guidelines:

- Основний колір: Темно-синій (#3F51B5), що нагадує креслення та технічні малюнки.
- Колір фону: Світло-сірий (#F0F2F5), забезпечує нейтральний фон, який не напружує очі при тривалому використанні.
- Колір акценту: Яскраво-помаранчевий (#FF5722) для кнопок із закликом до дії та для виділення ключової інформації.
- Шрифт для тексту та заголовків: 'Inter', sans-serif, сучасний, технічний, об'єктивний, нейтральний шрифт.
- Використовуйте чіткі, мінімалістичні іконки для представлення різних інструментів та функцій, з акцентом на впізнаваність та легкість розуміння.
- Чистий, структурований макет з чіткою ієрархією для полегшення навігації та отримання інформації.
- Тонкі переходи та анімації для забезпечення зворотного зв'язку та покращення користувацького досвіду без відволікання.
